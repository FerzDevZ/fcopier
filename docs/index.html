FCOPIER.

High-precision framework for website analysis, forensic cloning, and automated security auditing. Framework presisi tinggi untuk analisis situs web, cloning forensik, dan audit keamanan otomatis.

Technical Services Getting Started

01. Installation

Ensure you have Node.js 18+ installed on your system. Run the installer to initialize the environment. Pastikan Node.js 18+ terinstal di sistem Anda. Jalankan installer untuk inisialisasi environment. 

git clone https://github.com/FerzDevZ/fcopier.git
cd fcopier
bash install.sh

02. Core Cloning

Granular data extraction with proxy support and intelligent sync capabilities. Ekstraksi data granular dengan dukungan proxy dan fitur sinkronisasi cerdas. 

# Basic clone execution
fcopier clone https://example.com

# High-stealth mode through Proxy
fcopier clone https://example.com --proxy http://proxy.host --ua "Browser Agent"

03. Audit & Security

Automated forensic scanning to identify credential leaks (AWS, GitHub, Slack) and PII. Pemindaian forensik otomatis untuk identifikasi kebocoran kredensial dan paparan data sensitif. 

# Scan an existing directory
fcopier scan ./cloned_site

# Integrated real-time audit
fcopier clone https://example.com --scan

04. Exporting

Transform live web pages into professional documentation assets. Transformasi halaman web menjadi aset dokumentasi profesional. 

# PDF Document Export
fcopier export https://example.com --type pdf

# Portable Single-File HTML
fcopier export https://example.com --type singlefile

05. API Mocking

Simulate captured network traffic locally with the built-in enterprise dashboard. Simulasi lalu lintas jaringan lokal melalui dashboard enterprise yang terpasang. 

# Launch server with dashboard
fcopier serve ./cloned_site --open
